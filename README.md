# skills

Personal agent skills. Readability-first TypeScript, JavaScript, and JSX
rules that apply to every project.

## Install

npx skills add -g doubleoevan/skills -a claude-code -a opencode -a cursor -a codex -a github-copilot -a windsurf

## Update flow

1. Edit rules here, and only here. This repo is the source of truth.
2. `npx skills update` on each machine picks up changes.
3. Projects that vendor a pinned copy bump it deliberately.
